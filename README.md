## 📜 About

**The One Quiz** is a terminal-based Lord of the Rings quote trivia game. You're shown a real quote pulled live from the LOTR API — and you have to guess which film it came from.

10 rounds. May the Valar guide you.

Built on the [Jackson Blacklock SDK](https://github.com/jhblacklock/Jackson-Blacklock-SDK).

---

## 🚀 Setup

**1. Clone the repo**
```bash
git clone https://github.com/jhblacklock/The-One-Quiz.git
cd The-One-Quiz
```

**2. Create a virtual environment and install dependencies**
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

**3. Get a free API key** at [the-one-api.dev](https://the-one-api.dev)

**4. Play**
```bash
LOTR_API_KEY=<your-key> python game.py
```
